初次docker使用记录

2017-06-21

Docker基本操作

拉取镜像

docker pull hub.c.163.com/library/jenkins

查看本地镜像

xindeMacBook-Pro-4:~ xin$ docker images
REPOSITORY                      TAG                 IMAGE ID            CREATED             SIZE
hub.c.163.com/library/mysql     5.5.56              157f9c7bf6c6        2 months ago        256MB
hub.c.163.com/library/jenkins   latest              36cdf943d844        2 months ago        714MB

删除指定镜像

docker rmi image-id

删除所有的镜像

docker rmi $(docker images -q)

容器基本操作

（Docker支持将软件编译成一个镜像(image),在这个镜像中对软件做好各种配置,然后发布这个镜像,使用者可以运行这个镜像,运行中的镜像称之为容器）

运行镜像为容器

xindeMacBook-Pro-4:~ xin$ docker run -d --name jenkins -p 9090:8080 hub.c.163.com/library/jenkins
96de5761fe645d4cb4454da49aa18153d3e8f3a4ecfd317b762aff00f9461728
-d 代表终端关了还有在后台运行
—name 容器名
-p 端口

查看在运行的容器

xindeMacBook-Pro-4:~ xin$ docker ps
CONTAINER ID        IMAGE                           COMMAND                  CREATED             STATUS              PORTS                 NAMES
d5aaaba7b430        hub.c.163.com/library/jenkins   "/bin/tini -- /usr..."   40 seconds ago      Up 39 seconds       8080/tcp, 50000/tcp   android-jenkins

查看所有容器状态

xindeMacBook-Pro-4:~ xin$ docker ps -a
CONTAINER ID        IMAGE                                COMMAND                  CREATED             STATUS                   PORTS                 NAMES
d5aaaba7b430        hub.c.163.com/library/jenkins        "/bin/tini -- /usr..."   2 minutes ago       Up 2 minutes             8080/tcp, 50000/tcp   android-jenkins
d84dc40e2d5e        hub.c.163.com/library/mysql:5.5.56   "docker-entrypoint..."   2 weeks ago         Exited (0) 11 days ago                         mysql

查看镜像版本

xindeMacBook-Pro-4:~ xin$ docker run hub.c.163.com/library/jenkins --version
2.46.2

启动容器

xindeMacBook-Pro-4:~ xin$ docker start android-jenkins
android-jenkins

关闭容器

xindeMacBook-Pro-4:~ xin$ docker stop android-jenkins
android-jenkins

删除单个容器

docker rm 容器的id
删除所有的容器:docker rm ${docker ps -a -q}

查看容器日志

xindeMacBook-Pro-4:~ xin$ docker logs android-jenkins

登录容器 执行命令

xindeMacBook-Pro-4:~ xin$ docker exec -t mysql ls
bin   docker-entrypoint-initdb.d  home     media    proc  sbin  tmp
boot  entrypoint.sh          lib     mnt    root  srv   usr
dev   etc              lib64  opt    run   sys   var

jenkins初始密码
